\## 📋 Overview



\*\*Campus Buddy\*\* is a comprehensive college management system that helps students organize their academic life. Track classes, manage assignments, record marks, and calculate GPA - all in one place!



Perfect for students who want to stay organized and on top of their studies.



\---



\## ✨ Features



\### 📚 \*\*Class Management\*\*

\- ✅ Add new classes with professor details and timing

\- ✅ View all classes organized by subject

\- ✅ Search classes by subject name

\- ✅ Edit class information

\- ✅ Delete classes



\### 📝 \*\*Assignment Tracking\*\*

\- ✅ Add assignments with deadlines

\- ✅ Track assignment completion status

\- ✅ Search assignments by subject or keyword

\- ✅ Sort assignments by deadline or status

\- ✅ Mark assignments as complete

\- ✅ Edit assignment details

\- ✅ Delete assignments



\### 📊 \*\*Marks \& GPA Management\*\*

\- ✅ Record marks for each subject

\- ✅ Calculate percentages automatically

\- ✅ View all recorded marks

\- ✅ Calculate overall GPA (4.0 scale)

\- ✅ Get letter grades (A+, A, B, C, D)

\- ✅ Delete marks records



\### 💾 \*\*Data Export\*\*

\- ✅ Export all data to a text file

\- ✅ Generate professional reports

\- ✅ Includes classes, assignments, marks, and GPA summary

\- ✅ Timestamped export files
